# WizShell

WizShell is a PowerShell-based tool designed to aid in penetration testing. With a wide range of pre-built scripts, WizShell provides users with a library of command that can be used to automate various tasks involved in penetration testing. The tool is designed to be easy to use, with a user-friendly interface that simplifies the process of running and managing scripts.

# Locally run

```./wizshell.ps1```

# Remotely run

```powershell –nop –c “iex(New-Object Net.WebClient).DownloadString('https://raw.githubusercontent.com/4K3YM4L/WizShell/main/wizshell.ps1')”```
